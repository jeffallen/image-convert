Convert to PNG

Got an image in a format that Go knows? Get it converted to PNG!

The formats this tool knows are:

  • GIF (from the Go standard library)
  • JPEG (from the Go standard library)
  • BMP (from code.google.com/p/go.image/bmp)
  • TIFF (from code.google.com/p/go.image/tiff)
  • PCX (from github.com/ftrvxmtrx/groke/image/pcx)
  • XBM (from github.com/knieriem/g/image/xbm)
  • PNM (from github.com/knieriem/g/image/pnm)
  • CL2 (from github.com/mewrnd/blizzconv/images/cl2)
  • CEL (from github.com/mewrnd/blizzconv/images/cel)
Of course, you can also change a PNG into a PNG, if you really want.
